# Инструкция по серверной отправке заявок (PHP)

Публичный блок загрузки каталога из Excel удален из интерфейса сайта, чтобы посетители не могли менять каталог.

## 1. Что уже сделано
- Форма на сайте отправляет `POST` запрос в `send_request.php`
- Сервер валидирует `ФИО` и `Телефон`
- При успешной отправке возвращается JSON `success: true`

## 2. Обязательная настройка
Откройте `send_request.php` и замените:
- `$to = 'sales@img-company.ru';` на корпоративный email компании
- `$from = 'noreply@img-company.ru';` на адрес вашего домена

## 3. Требования к хостингу
Нужен PHP-хостинг с работающей функцией `mail()` или SMTP-шлюзом.

Если `mail()` у провайдера отключен, используйте SMTP (PHPMailer) вместо встроенной `mail()`.

## 4. Локальный запуск
Для проверки через PHP-сервер:
1. Откройте терминал в папке проекта.
2. Выполните: `php -S localhost:8000`
3. Откройте сайт: `http://localhost:8000/main.html`

## 5. Валидация заявки
- ФИО: 5-100 символов, буквы/пробел/дефис
- Телефон: `+7XXXXXXXXXX` или `8XXXXXXXXXX`

## 6. Excel-шаблон
Файл `products-template.xlsx` оставлен в проекте как шаблон для внутренней работы с каталогом.
